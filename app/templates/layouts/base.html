Как работает сервис

Поиск товара-аналога по артикулу

По заданному артикулу система находит товар-аналог другого производителя. Поиск происходит с учетом технических характеристик товара.

На кого рассчитан наш сервис

Если вы электромонтажная организация, проектировщик или производитель, и вам необходимо быстро «перевести спецификацию с одного производителя на другого».

В каких товарных категориях работает сервис

Пока поиск осуществляется в категории «кабеленесущие системы» (кабельные лотки). Количество представленных производителей и товарных категорий, среди которых выполняется поиск, постоянно расширяется.

Если Вы хотите помочь нам улучшить работу сервиса

Если вы не нашли требуемого производителя, напишите, пожалуйста, на наш адрес – user@analogpro.ru Мы ускорим работу по данному производителю и сообщим вам о готовности. Поиск постоянно совершенствуется, но если Вы заметили неточности в работе поиска, большая просьба сообщить нам об этом по адресу - user@analogpro.ru для устранения неточностей.

Поиск по артикулу

Введите исходные данные

Укажите артикул товара, для которого надо найти аналог. Выберите производителя По заданному артикулу система находит товар-аналог другого производителя. Поиск происходит с учетом технических характеристик товара.

{#
#}

Результат

В окне результата Вы получите артикул товара-аналога, максимально соответствующего по своим свойствам исходному товару. Нажав на соответствующую кнопку, можно увидеть подробные технические характеристики.

Суперпоиск

Система позволяет изменить параметры поиска с учетом дополнительных требований пользователя. Для этого необходимо воспользоваться кнопкой «Открыть расширенную форму» и выбрать соответствующие параметры. В этом случае будет найден товар, максимально соответствующий заданным характеристикам.

Поиск по спецификации

Поиск сразу нескольких артикулов

Функция позволяет найти товары-аналоги сразу для списка исходных артикулов. Для этого необходимо выбрать исходный файл. Файл должен быть в формате EXCEL(xls или xlsx), в котором первый столбец должен содержать артикулы, по которым будет проводиться поиск.

Результат

Результаты поиска будут сохранены в файле. Файл будет отправлен на электронный адрес, указанный при регистрации. Для пользования функцией регистрация обязательна. Регистрация бесплатная.
